# Végétal Restaurant CLI

A Java console application simulating a restaurant ordering system 
with invoice generation.

## Features
- Interactive menu with 6 items
- Multi-item ordering with quantities
- Automatic discount: 5% over $50, 10% over $100
- Quebec tax calculation (14.975%)
- Order summary at end of session
- Multiple orders per session support

## Tech Stack
Java, Scanner (console I/O)

## How to Run
javac Tp1Prog.java
java Tp1Prog
